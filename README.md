# Fidelity Optimizer

A local, privacy-first tool designed to analyze your Fidelity portfolio, identify tax loss harvesting opportunities, and recommend tax-efficient alternative funds mathematically modeled for each account type. 

## Getting Started
It's designed to be extremely easy to use:
1. Drop your financial CSVs (and 401k NetBenefits PDF) into the `Drop_Financial_Info_Here/` folder.
2. If you have a 401k PDF, simply double-click the `Fidelity_401k_PDF_Extractor.bat` file once.
3. Double-click the main `Fidelity_Optimizer.bat` file.
4. Your analysis will automatically pop open as a PDF!

For more detailed instructions, please read [How to Use](docs/HOW_TO_USE.md).

Curious about how the math, scoring, and tax routing actually works behind the scenes? Read our plain-English guide: [How it Works (For Non-Technical Users)](docs/HOW_IT_WORKS.md).
